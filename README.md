# Midnight Glass

Replacement for the default chat. Smooth as glass!

![Imgur](https://i.imgur.com/8lj13ch.gif)

## Options

Use **`/mglass`** or **`/mg`** to open the in-game config.

**`/mgdebug`** for the debug mode.
